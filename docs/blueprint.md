# **App Name**: WodMatch

## Core Features:

- Athlete Registration: Allow athletes to register via email or Google, providing necessary details like ID, phone, gender, location, box affiliation, and coach information.
- Competition Marketplace: Enable organizers to list events with details like banners, dates, location, and descriptions for athletes to browse and register.
- Partner Finder: Create a space within team or pairs competitions where athletes can find partners, viewing profiles, times, and PRs. LLM will use these metrics as a tool in helping athletes to find ideal partners.
- Payment Management: Allow athletes to pre-register for competitions and upload proof of payment. Organizers can validate and approve payments, updating the athlete's status accordingly.
- Automated Leaderboard: Implement an automatic scoring system based on times or reps loaded, including tie-breaker rules, and a fast interface for judges to input scores.
- Notifications System: Automate notifications (push/email) for payment confirmations/rejections and competition updates.
- Admin Event Creation: Enable organizers to create events with detailed settings like dates, categories, prices, and regulations (uploadable as PDF/Doc).

## Style Guidelines:

- Primary color: Neon green (#39FF14) to represent energy, action, and the CrossFit community.
- Background color: Dark gray (#1A1A1A) to create a premium, high-performance aesthetic that highlights the green.
- Accent color: Electric blue (#7DF9FF) for interactive elements and key CTAs.
- Headline font: 'Space Grotesk' (sans-serif) for a computerized, techy, high-performance feel. Body font: 'Inter' (sans-serif) for longer text.
- Use sharp, modern icons related to CrossFit movements and competition elements.
- Design cards and buttons optimized for thumb use on smartphones, keeping a mobile-first approach.
- Incorporate loading spinners and empty states to ensure a smooth user experience and clear feedback.
